# LifeOS UI 重新设计方案

基于 ui-ux-pro-max 设计系统

## 设计方向

### 视觉风格
- **色系**: 深色优先，靛蓝为主色调
- **风格**: 现代简约，卡片式布局
- **氛围**: 专业、高效、温暖

### 核心组件

#### 1. 导航栏
- 渐变背景 (indigo-500 to purple-600)
- 玻璃态效果
- 响应式设计

#### 2. 卡片
- 圆角 16px
- 悬停阴影效果
- 微交互动画

#### 3. 按钮
- 主按钮：渐变背景
- 次按钮：边框样式
- 动画：scale 0.98 on click

### 调色板

#### 主色系
- Primary: indigo-500 (#6366f1)
- Secondary: purple-500 (#a855f7)
- Accent: pink-500 (#ec4899)

#### 中性色
- Surface: #1e1e2e (深色)
- Surface-elevated: #2a2a3c
- Text-primary: #e4e4e7
- Text-secondary: #a1a1aa

### 排版
- 标题：30px / 40px / 48px
- 小标题：18px / 20px / 24px
- 正文：14px / 16px
- 辅助：12px / 13px

### 间距系统
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
